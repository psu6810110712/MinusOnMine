## 📖 เกี่ยวกับเกม

**MinusOnMine** เป็นเกมแนว Top-Down Mining RPG ที่คุณจะได้รับบทเป็นนักขุดแร่ ลงพื้นที่เพื่อทลายก้อนหิน ค้นหาอัญมณีมีค่า และนำเงินมาอัปเกรดอุปกรณ์เพื่อทำลายสถิติของตัวเอง

### ลูปการเล่นหลัก (Core Gameplay Loop)

1. **ขุดแร่ (Mine):** เดินสำรวจเหมืองและเข้าประชิดก้อนหินเพื่อทุบ หินแต่ละชนิดมีความแข็ง (HP) และให้เงินรางวัล (Coins) ที่แตกต่างกัน
2. **จัดการเส้นทาง (Pathing):** ตัวละครสามารถเดินได้เฉพาะบน "พื้นที่ว่าง" เท่านั้น คุณต้องทุบหินเพื่อเปิดทางและสร้างเส้นทางขุดที่คุ้มค่าที่สุด
3. **อัปเกรด (Upgrade):** นำเงินที่เก็บสะสมได้ไปใช้งานที่ "ร้านค้า (Shop)" ทางด้านขวาของหน้าจอ เพื่อซื้อการอัปเกรดสเตตัส เช่น:
   - **Power:** เพิ่มพลังทำลายล้าง ทำให้ทุบหินแข็งๆ แตกได้ไวขึ้น
   - **Wealth:** เพิ่มอัตราตัวคูณเงินรางวัลที่ได้จากการทุบหิน

## 🎮 วิธีการควบคุม

| ปุ่ม | ฟังก์ชัน |
|-----|---------|
| **W, A, S, D** | บังคับตัวละครให้เดิน (ขึ้น, ซ้าย, ลง, ขวา) |

## 🛠️ ความต้องการของระบบ

ก่อนรันเกม MinusOnMine โปรดตรวจสอบว่าติดตั้งสิ่งต่อไปนี้แล้ว:

- **Python 3.8 ขึ้นไป**
- **pip** (Python package manager)

## 📦 วิธีการติดตั้ง

### ขั้นตอนที่ 1: โคลนหรือนำทางไปยังโปรเจกต์

```bash
cd MinusOnMine-main
```

### ขั้นตอนที่ 2: สร้าง Virtual Environment (ขอแนะนำ)

```bash
# บน Windows
python -m venv venv
venv\Scripts\activate

# บน macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### ขั้นตอนที่ 3: ติดตั้ง Dependencies

```bash
pip install -r requirements.txt
```

หรือติดตั้ง Kivy โดยตรง:

```bash
pip install kivy
```

## 🚀 วิธีรันเกม

เมื่อติดตั้ง dependencies เรียบร้อยแล้ว ให้รันเกมด้วยคำสั่ง:

```bash
python main.py
```

หน้าต่างเกมจะปรากฏขึ้น และคุณสามารถเริ่มเล่นได้ทันที!

## 🔧 การแก้ไขปัญหา

### ข้อผิดพลาด "Kivy not found"
ตรวจสอบว่าคุณได้เปิดใช้งาน virtual environment และติดตั้ง dependencies:
```bash
pip install kivy
```

### ข้อผิดพลาด Permission denied บน Linux/Mac
ตรวจสอบว่าไฟล์มีสิทธิ์ในการทำงาน:
```bash
chmod +x main.py
```

### ปัญหาการแสดงผลบนหน้าจอ
ลองรันกับแบ็กเอนด์กราฟิกส์ที่เฉพาะเจาะจง:
```bash
python main.py
```
