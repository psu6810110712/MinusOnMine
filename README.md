# ⛏️ MinusOnMine

**เกมแนว Top-Down Mining RPG พัฒนาด้วย Kivy**

MinusOnMine คือโปรเจกต์เกมขุดแร่แบบมุมมองด้านบนบนแผนที่ Grid ผู้เล่นต้องวางแผนเส้นทางการขุด เก็บทรัพยากรให้คุ้มที่สุด และนำเงินไปอัปเกรดเพื่อทำคะแนนให้ดีขึ้นในแต่ละรอบ

---

## 📌 สถานะโปรเจกต์
- เวอร์ชันปัจจุบัน: `v0.1.x`
- ส่วนที่ทำแล้ว: รูปแบบเกม, Gameplay, ภาพรวมของเกม
- ส่วนที่กำลังต่อ: ตัวละคร ไอเทม visual ต้างๆ โครงสร้างแอปหลัก, เมนูเริ่มเกม, โครงหน้าเล่น, เอกสารระบบ ระบบขุดเ, ระบบเงินและเลเวลในเกม, Save/Load จริง

---

## 🎮 Core Gameplay Loop

1. **Mine**: เดินและทุบหินในแผนที่ตาราง
2. **Collect**: รับรางวัลตามประเภทแร่และความยากของก้อนหิน
3. **Upgrade**: นำเงินไปเพิ่มความสามารถผู้เล่น
4. **Optimize**: วางแผนการขุดรอบต่อไปให้คุ้มค่ามากขึ้น

---

## ✨ ขอบเขตระบบ
- ระบบการเคลื่อนที่แบบ Grid
- ระบบ Interact/ทุบหิน
- ระบบเงินและการคำนวณรางวัล
- ระบบอัปเกรดผู้เล่น (เช่น Power, Wealth. Speed, View Distance)
- ระบบจัดการสถานะเกม (GameState)
- ระบบ Save/Load ด้วย JSON
- UI ด้วย Kivy + `.kv` language

## ✅ ความจำเป็นของงานหลัก

- **ระบบขุดและรางวัล**: เป็นหัวใจของเกม ถ้าไม่เสร็จเกมจะไม่มี loop การเล่น
- **ระบบอัปเกรด**: ทำให้เกิดแรงจูงใจในการเล่นต่อ และเห็นความก้าวหน้า
- **ระบบบันทึกข้อมูล**: จำเป็นสำหรับการทดสอบระยะยาวและเดโมหลายรอบ
- **UI/Feedback**: ช่วยให้ผู้เล่นเข้าใจสถานะเกมและลดความสับสนตอนเล่นจริง
- **เอกสาร README/Docs**: ใช้สื่อสาร architecture และการแบ่งงานให้ตรวจประเมินได้ชัดเจน
