# ⛏️ MinusOnMine

**Top-Down Mining RPG พัฒนาด้วย Kivy**

โปรเจกต์เกมขุดแร่แบบมุมมองด้านบน (Grid-based) ผู้เล่นรับบทนักขุดแร่ สำรวจเหมือง ทุบหิน เก็บทรัพยากร สะสมเงิน และอัปเกรดความสามารถเพื่อทำสถิติใหม่ของตัวเอง

## 🎮 Core Gameplay

1. **Mine**: เดินสำรวจแผนที่ตารางและทุบหินที่อยู่ติดตัวละคร  
2. **Collect**: รับเงิน/ทรัพยากรจากหินแต่ละประเภท  
3. **Upgrade**: ใช้เงินซื้ออัปเกรดเพื่อขุดไวขึ้นและได้ผลตอบแทนมากขึ้น  
4. **Repeat**: วางแผนเส้นทางขุดเพื่อทำกำไรสูงสุดในแต่ละรอบ

---

## ✨ Features

- ระบบเกมแบบ Grid Movement
- ระบบทุบหินและคำนวณค่ารางวัล
- ระบบอัปเกรดผู้เล่น (เช่น Power, Wealth)
- โครงสร้าง GameState สำหรับจัดการสถานะเกม
- ระบบ Save/Load ผ่านไฟล์ JSON
- UI พัฒนาด้วย Kivy + `.kv` language
